================================================================================

Flash Lite: Sudoku Example

================================================================================

 

This example demonstrates how to design a simple Flash Lite Sudoku game, add dynamic layout control for multiple screen resolutions, and use several input methods (key, touch, and key & touch).

 

 

CONTENTS OF THE PACKAGE

-------------------------------------------------------------------------------

SudokuMaster.fla:  Adobe Flash Professional CS3 project file

board.as: ActionScript file

mytimer.as: ActionScript file

optionsmenu.as: ActionScript file

release_note.txt: This file

 

 

USING THE APPLICATION

-------------------------------------------------------------------------------

The application has only one view.

 

Keys:

- Use navigation keys to navigate through the Sudoku board.

- Use the keypad to enter a number into a cell.

- Use the Select key to open the softpad and select a number.

           

Touch:

- Tap and hold to navigate the Sudoku board.

- Release to select and enter a number.

-You can also tap a cell to select and enter a number.

 

 

BUILD & INSTALLATION INSTRUCTIONS

-------------------------------------------------------------------------------

1.  Open SudokuMaster.fla using Adobe Flash Professional CS3 (or newer).

2.  Compile the project and test it using Adobe Device Central.

3.  Send and execute the SudokuMaster.swf file to a device supporting Flash Lite 2.x.

 

 

COMPATIBILITY

-------------------------------------------------------------------------------

All devices supporting Flash Lite 2.x and higher (S60 3rd Edition, FP1 and newer, Series 40 3rd Edition FP2 and higher).

 

Tested on Nokia 5800 XpressMusic, Nokia E90 Communicator, Nokia N95, Nokia N95 8GB, Nokia N76,

Nokia 7510 Supernova, and Nokia E71.
