压缩包中有三个文件夹，分别是SWF Decompiler、旧版本的mashup-editor源码、我收集的可能用到的素材。
第一个文件夹中包含SWF文件反编译器。
第二个文件夹中包含旧版本的（当前在服务器上运行的版本）mashup-editor。
第三个文件夹中包含我这几天收集和测试过的Flash素材，主要是一些能够用在我们项目中的按钮。所有的SWF我都亲自测试过。
文件mashup-editor.fla是我绘制的一个按钮，之所以亲自绘制，是因为素材中的按钮在扩展性上存在一些问题。
文件rainy.png是测试mashup-editor时用到的图标文件，无关紧要。

每个文件夹中都还有一个readme.txt，详细说明了该文件夹的相关情况。

我这几天的工作主要是对新的素材进行测试，以及设计新的UI，我把我的想法大概说一下：

1、进入界面显示出的是一个类似于iPhone界面的Most Popular Blocks,显示的是目前使用最多的block（具体的界面我已经制作出来，但是没有事件关联，我觉得使用价值不大，因此没有一并发给你）。

2、在上面提到的界面里，block的图标就显得很重要了，它是标识block的符号，在mashup-editor.fla里，我做的就是这个工作，rainy.png就是我作为图标的图片（当然图标涉及到整个block xml定义的更改，可以暂时搁置，否则工作量大的惊人）。

3、目前主要的工作就是把一个类似于iPhone的界面和我们之前的界面结合起来，之前的界面已经有了，只要制作出一个类似于iPhone的界面并能够正确工作和与之后的界面通信就可以了。
